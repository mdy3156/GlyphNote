# 04 MVP Roadmap

## Phase 1: Vault とノート基盤
- Vault を開く/作る
- `.tex` / `.typ` ノート新規作成
- 保存・再読込

## Phase 2: 探索性
- タグ管理
- 全文検索
- 更新日時ソート

## Phase 3: 知識接続
- `[[WikiLink]]` パース
- バックリンク表示
- ノート間遷移

## Phase 4: 出力
- LaTeX / Typst 手動ビルド
- PDF プレビュー
- ビルドログ表示

## 完了基準
- 大学院生が1つの研究テーマのノートを作成し、1週間運用して再利用性を実感できる
