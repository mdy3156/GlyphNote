# 00 Product Vision

## 目的
GlyphNote は、LaTeX / Typst を主役にした研究ノートアプリである。大学院生や研究者が、理論メモ・定理・証明・導出を継続的に書き、後からすぐ再利用できる状態を作る。

## 対象ユーザー
- LaTeX または Typst でノートを書く人
- 数学・理論系の研究で、導出や定理整理を日常的に行う人
- フォルダ整理を自動化し、タグやリンクで探索したい人

## 提供価値
- 記述言語を妥協しない: Markdown の補助ではなく LaTeX / Typst を中心に扱う
- 再発見しやすい: タグ、リンク、バックリンク、全文検索で既存ノートを再利用しやすくする
- 継続しやすい: 定理/証明テンプレートや導出ノート形式で記録負荷を下げる

## 非目標（初期）
- 汎用ドキュメント編集（Notion 的な万能性）
- クラウド共同編集
- 高度なリアルタイム同期

## MVP 成功条件
- 1つの Vault で .tex/.typ ノートを作成・編集・検索・再閲覧できる
- ノート間リンクとタグで辿れる
- 手動ビルドで PDF を安定生成できる
